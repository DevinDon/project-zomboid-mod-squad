# 战术小队 Squad

**Project Zomboid B42+ 模组** — 在末日中组建并指挥你的精英小队！

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 📖 简介

**战术小队 Squad** 是一款为 Project Zomboid B42+ 设计的模组，让玩家扮演小队长，在末日世界中招募 NPC 幸存者，组建自己的精英小队。

通过指令轮盘下达战术命令，队员会根据经验和技能自动成长，形成真正有战斗力的生存团队。

## ✨ 核心特性

### 🎯 招募与小队管理

- 在游戏世界中招募 NPC 幸存者加入你的小队
- 每个队员拥有独立的背包、状态、技能和经验值
- 队员会随着战斗和任务积累经验，自动升级成长
- 队员信息面板：查看健康、体力、背包、技能详情

### 📋 指令系统

- **指令轮盘** — 直观的放射状命令菜单
- **📍 跟随** — 队员保持阵型跟随队长
- **⚔️ 攻击** — 进攻指定目标或位置
- **🛡️ 防御** — 在指定区域巡逻警戒
- **🎒 搜刮** — 自动搜索并收集物资
- **✋ 停下** — 坚守当前位置
- **🛡️ 守卫据点** — 守卫指定地点

### 🧠 智能行为系统

- 基于状态机的行为程序（Program）系统
- 每个程序由多个阶段（Stage）组成，自动流转
- 支持任务队列（Task），队员会按序执行复杂指令
- 根据环境自动调整行为（战斗、撤退、搜刮等）

### 📈 经验与升级

| 行为 | 经验 | 相关技能 |
|------|------|----------|
| 击杀僵尸（近战） | 10-20 | 近战 |
| 击杀僵尸（射击） | 15-25 | 射击 |
| 跑步/负重移动 | 2-5 | 体力 |
| 潜行移动 | 3-8 | 潜行 |
| 治疗队友 | 10-15 | 医疗 |
| 战斗胜利 | 20-30 | 全技能 |

### 🌐 多人兼容

- 完整的客户端/服务端数据同步
- 使用 Project Zomboid ModData API 进行数据存储
- 32 个数据集群按队员 ID 哈希分布，保障多人性能

## 🎮 快速开始

### 安装

1. 从 Steam 创意工坊订阅本模组
2. 在游戏启动器的 Mod 列表中启用 `[B42] 战术小队 Squad`
3. 进入游戏，在沙盒设置中可以调整初始队员数量等参数

### 基本操作

1. **招募队员** — 右键空地 → "招募队员"
2. **下达命令** — 右键队员 → "下达命令" → 选择指令
3. **查看信息** — 右键队员 → "查看信息"
4. **小队管理** — 右键自己 → "小队管理"
